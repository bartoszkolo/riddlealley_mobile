Native mobile port of the RiddleAlley PWA, built with Flutter.

## Overview
Urban escape quests in your pocket. This app handles:
- GPS tracking for geolocation-based puzzles
- Offline capabilities
- Native notifications
- Smoother animations and transitions

## Tech Stack
- **Framework:** Flutter
- **State Management:** Riverpod
- **Routing:** GoRouter
- **Backend:** Supabase
- **Maps:** flutter_map (OpenStreetMap based)

## Getting Started

1. **Prerequisites:**
   - Flutter SDK (latest stable)
   - VS Code or Android Studio

2. **Installation:**
   ```bash
   flutter pub get
   flutter run
   ```

## Project Structure
- `lib/src` - Feature-based architecture
- `lib/shared` - Shared components and utilities
